# Hello World – GitHub Pages

A fancy animated "Hello World" page served via GitHub Pages.

## Live site

Once GitHub Pages is enabled for this repository (Settings → Pages → deploy from branch `main` / root), the site will be available at:

```
https://cceryl.github.io/hello/
```
